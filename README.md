- Python 3.7.4 +
- Uses [Instalooter](https://github.com/althonos/InstaLooter.git)

# Usage:
```
pip install -r requirements.txt
python main.py
```
Browse to: `localhost:5000`

# Note:
- Only public accounts.
- All data is saved in dir `./data/<username>`
